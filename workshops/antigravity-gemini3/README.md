# Workshop Notes: Google Antigravity – Agentic Development Platform

## 1. What is Antigravity IDE?
- **IDE:** Integrated Development Environment
  - Allows users to write, debug, and run code
  - Popular examples: VS Code, PyCharm, Jupyter, Spyder

- **Recent IDEs used by companies:**
  - Cursor IDE
  - Cline
  - Google ADK
  - Kiro
  - Intelliject
  - Eclipse
  - **Antigravity IDE** – Google (released last week)

- **Why these IDEs are important:**
  - Helps understand trends like automation, agentic development, and productivity improvements

## 3. Antigravity IDE Installation
- Search for **“Antigravity Google IDE”** → Download → Install
- Click **Next** → Select **Workflow (Start Fresh)** → Next → Sign in with Gmail account

## 4. IDE vs Agentic AI IDE
| Feature | Traditional IDE | Agentic AI IDE |
|---------|----------------|----------------|
| Code Writing | User writes code | Agent writes code automatically |
| Automation | Minimal | High, built-in agents available |
| LLM Integration | None | Google Gemini model generates code |
| Example | VS Code, PyCharm | Antigravity IDE |

- **Key Concept:**
  - Agentic AI IDE allows developers to leverage AI agents that can **write, debug, and optimize code** for them
