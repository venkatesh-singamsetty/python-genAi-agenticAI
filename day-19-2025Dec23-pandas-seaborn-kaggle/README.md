# Matplotlib + Seaborn --- Notes

## Matplotlib + Seaborn

-   Used for data visualization in Python
-   Helps in EDA (Exploratory Data Analysis)
-   Matplotlib → base plotting library
-   Seaborn → built on Matplotlib, statistical & clean plots

------------------------------------------------------------------------

## Univariate Analysis

-   Plot graph using 1 variable
-   Understand data distribution
-   Identify spread and outliers
-   Common plots:
    -   Histogram
    -   Box plot
    -   Count plot

------------------------------------------------------------------------

## Bivariate Analysis

-   Plot graph using 2 variables
-   Understand relationship between two variables
-   Common plots:
    -   Scatter plot
    -   Line plot
    -   Bar plot
    -   Box plot

------------------------------------------------------------------------

## Multivariate Analysis

-   Plot graph using 3 variables
-   Analyze interaction among variables
-   Common plots:
    -   Pair plot
    -   Heatmap
    -   Facet plots

------------------------------------------------------------------------

## Outlier / Anomaly Detection

-   Outlier = data point very different from others
-   Used to identify abnormal data
-   Common methods:
    -   Box plot
    -   Statistical methods
